Боевые техники

Главная Передвижение

Rocket Surfing

Игрок выстреливает себе под ноги ракетницей, используя скольжение на ракете для перемещения на высоту или дальность.

Core Nuking

Создаёт мощный взрыв, комбинируя Core Eject и Railgun, чтобы нанести огромный урон.

Deadcoin

Метание монетки и выстрел в неё в правильный момент удваивает урон монеты.

Railcoin

Выстрел по монете из Railgun позволяет нанести урон сквозь препятствия с высокой точностью.

Projectile Boost

Удар по пулям дробовика, чтобы ускорить и усилить выстрел. Можно использовать с техникой Shotgun Swapping!

Shotgun Swapping

Чередование дробовиков для быстрого нанесения урона без потери темпа стрельбы. Можно использовать с техникой Projectile Boost!
